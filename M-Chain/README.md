## Installation

1. Clone the repository:

```bash
git clone https://github.com/molalign8468/WEB3.0-Alchemy-University-Bootcamp.git/M-Chain
cd M-Chain
```

2. Install dependencies:

```bash
npm install crypto-js
```

2. Install dependencies:

```bash
node index.js
```
